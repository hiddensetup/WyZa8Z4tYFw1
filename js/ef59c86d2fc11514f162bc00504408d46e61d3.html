Live Chat Setup Guide

  • Download the ZIP file:
    • Click the following link: Download
  • Extract the downloaded file:
    • Once the download is complete, extract the js.zip file into the root folder of your website where you plan to integrate the Live Chat.
  • Add the CSS link tag: 
    <link id="steambox" rel="stylesheet" type="text/css" href="https://mysteamboxpath.steamboxchat.com/css/main.css" media="all">
  • Add the JavaScript scripts at the end of the HTML file: 
    <script src="js/jquery.js"></script>
<script id="steambox" src="/js/main.js"></script>
  • Optional: Add voice support: 
    <script src="js/WebAudioRecorder.min.js"></script>
<script src="js/app.js"></script>
